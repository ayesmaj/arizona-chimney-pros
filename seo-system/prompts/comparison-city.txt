CONTENT INSTRUCTIONS:

1. SLUG
Format from primary keyword: {option-a}-vs-{option-b}-{service}-{city-slug}

2. TITLE (H1)
State the comparison naturally.
Example: "Gas vs Wood Fireplace in Arizona: Which Is Better?"
Include the city/state if a city was provided; otherwise use "Arizona".

3. META TITLE
Comparison + location + decision signal. Max 60 chars.
Example: "Gas vs Wood Fireplace AZ | Cost, Safety, Which Wins"

4. META DESCRIPTION
140-160 chars. Frame it as an honest comparison guide.

5. INTRO (150-200 words)
- Open by naming the decision the homeowner is facing
- Acknowledge it's a real trade-off — both options have fans
- Preview what this page covers (cost, maintenance, safety, local fit,
  recommendation)
- Mention {{city}} or Arizona once
- No generic "let's explore..." openings

6. OPTION A SECTION (200-250 words)
Deep dive on Option A. Structure:
- One paragraph introduction ("What it is / how it works")
- <ul> of 4-5 PROS
- <ul> of 3-4 CONS
Be honest — don't stack the deck.

7. OPTION B SECTION (200-250 words)
Same structure as Option A.
- Intro paragraph
- <ul> of 4-5 PROS
- <ul> of 3-4 CONS
Match the depth — don't make one option look obviously worse.

8. COMPARISON TABLE SECTION (180-230 words)
Side-by-side facts. Open with a short intro paragraph.
Render an HTML <table> with these columns:
- "Factor"
- "Option A" (name it)
- "Option B" (name it)

Rows (adapt to the comparison):
- Upfront cost
- Installation cost
- Operating cost (per use)
- Heat output
- Efficiency (%)
- Maintenance frequency
- Typical lifespan
- Aesthetic appeal
- Insurance impact
- Smell / smoke / mess

At least 6-8 rows. Use actual numbers where possible (e.g., "$800-1,400"
not "variable").

Close with a one-sentence summary of what the table shows.

9. COST DIFFERENCE SECTION (160-210 words)
Focus purely on money. Cover:
- Upfront installation cost difference (typical range)
- Annual operating cost (gas bill vs wood cost vs electricity)
- Maintenance cost (annual inspection, chimney sweep vs gas tune-up)

Be specific. Use ranges. Reference {{price_range}} if relevant.

10. MAINTENANCE SECTION (150-200 words)

Arizona household.
